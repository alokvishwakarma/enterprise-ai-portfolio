## Add the Safety Inspection live URL
Open `index.html`, search for `Add Live Demo URL`, replace `href="#"` with the actual Streamlit URL, change the label to `Open Live Demo`, and remove `disabled` from the class.

## Deploy with GitHub Pages
1. Create a public GitHub repository named `enterprise-ai-portfolio`.
2. Upload these files.
3. Open the repository's Settings.
4. Select Pages.
5. Choose `Deploy from a branch`.
6. Select `main` and `/ (root)`.
7. Save.

https://alokvishwakarma.github.io/enterprise-ai-portfolio/
